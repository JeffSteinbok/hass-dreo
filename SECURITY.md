# Security Policy

Thank you for helping keep **hass-dreo** and its users safe.

This is an unofficial, community-maintained Home Assistant custom integration for
Dreo smart devices. It is not affiliated with or endorsed by Dreo. Please keep that
scope in mind when reporting issues — vulnerabilities in Dreo's cloud service,
firmware, or mobile apps should be reported directly to Dreo.

## Supported Versions

Security fixes are provided for the current releases only. Older versions are not
patched — please update before reporting.

| Version                 | Supported          |
| ----------------------- | ------------------ |
| Latest stable release   | :white_check_mark: |
| Latest beta release     | :white_check_mark: |
| Any older release       | :x:                |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**
Public issues can expose users before a fix is available.

Instead, report privately by reaching out to the maintainer directly:

- **Discord** or **LinkedIn** — contact details are listed on my GitHub profile:
  [@JeffSteinbok](https://github.com/JeffSteinbok)

When reporting, please include as much of the following as you can:

- A description of the vulnerability and its potential impact
- Steps to reproduce (proof-of-concept, affected device model, or logs if relevant)
- The version of the integration you're running
- Any suggested remediation, if you have one

Please **redact credentials, tokens, and other sensitive data** from any logs or
diagnostics you share.

## What to Expect

- I'll acknowledge your report as soon as I reasonably can.
- I'll investigate and keep you updated on progress toward a fix.
- Once a fix is released, I'm happy to credit you for the discovery — let me know if
  you'd prefer to remain anonymous.

Thanks for reporting responsibly. 🙏
