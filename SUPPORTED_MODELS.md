## Model Not Listed?

If you have a Dreo device that's not listed here, please [open an issue](https://github.com/JeffSteinbok/hass-dreo/issues) with the following information:

- Model number (in the format above)
- Device type (fan, heater, etc.)
- Number of speeds supported (for fans)
- Whether oscillation is supported
- What preset modes are available
- Any special features

See the [Adding New Devices](#adding-new-devices) section in the main README for more details on how to help add support for your device.
